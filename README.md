# Dungeon Scaling



This project originated as a backport of the mod **[Dungeon Difficulty](https://github.com/ZsoltMolnarrr/DungeonDifficulty)** by [**Daedalus** (daedaulus_dev/ZsoltMolnarrr)](https://github.com/ZsoltMolnarrr) for **Minecraft 1.20.1**. It has since diverged significantly and is now developed
and maintained as an independent project.

All credit for the original concept, design, and code goes to the original author.
